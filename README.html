The Methodology

With 112 heroes yielding over 4.5 million possible team combinations, manual synergy analysis during a live draft is impossible. We framed this as a probabilistic classification problem.

We constructed two feature vectors, Fwin and Floss, where x ∈ ℝ112. A value of 1 represents a chosen hero, and 0 represents an unselected hero. By dividing our 50,000 match dataset into a 70/30 split, we calculated the prior probability and the likelihood of winning with a particular hero.

01

Data Preprocessing

Extracted 50,000 international matches from Kaggle (Valve API), constructing sparse matrices of 50,000 rows by 112 hero columns.

02

Probability Modeling

Calculated the prior (0.5) and smoothed log-likelihoods for each hero derived from historical win/loss distributions.

03

Real-time Scoring

For a given partial team, the engine evaluates ~80,000 valid completions by summing log-posteriors, instantly returning top recommendations.
